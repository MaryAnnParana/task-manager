# 📋 Task Manager API

## 📝 Project Description
This project is a fully functional Laravel-based Task Management application designed to help users organize, track, and categorize their daily tasks. It features a relational database structure with clean controllers handling task logic, and migration files creating both categories and tasks tables with proper foreign key constraints.
